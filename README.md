# RentKart

*Why buy when you can rent... just RentKart*

RentKart is a full-stack rental platform built using the *MERN stack (MongoDB, Express.js, React.js, Node.js)*.  
It allows users to rent *electronics, furniture, bikes, scooters, ACs, coolers, inverters, and more* on a *daily, weekly, or monthly basis* â€” all from a single, user-friendly platform.

---

## Features

### User Features
- Register & Login (JWT Authentication + Google OAuth)
- Browse and filter items by *category* and *location*
- Add items to *Cart* or *Wishlist*
- Place rental orders with flexible durations
- Manage *Orders, **Returns, and **Reviews*
- Update personal *Profile* and *Addresses*

###  Owner/Admin Features
- Add, update, and remove item listings
- View and manage rental requests
- Owner dashboard to view rentals and earnings
- Admin authentication for secure access
- Approve or reject return requests

###  Platform Features
- Secure *JWT-based authentication*
- *Slug URLs* for SEO-friendly item pages
- *Dynamic cart* and *checkout system*
- *Security deposits* and *penalty fee system*
- *Wishlist, *notifications
- *Scheduled delivery and pickup*
